## 📌 Featured Projects

### AI Gesture & Sign Language Interaction System

A real-time AI application capable of recognizing hand gestures and sign language using Computer Vision and Deep Learning technologies. The system enables touchless interaction through gesture recognition and translates sign language into text in real time.

**Technologies**

- Python
- OpenCV
- MediaPipe
- TensorFlow
- PyTorch
- Scikit-learn

---

### AI-based Durian Sales Forecasting System

A machine learning project developed to forecast future durian sales using historical business data. The system provides sales predictions, interactive dashboards, and business insights to support inventory and production planning.

**Technologies**

- Python
- Pandas
- Scikit-learn
- XGBoost
- Plotly
- MySQL

---

### V-SPACE

An industrial AI inspection system developed during my internship at Seagate Technology. The system utilizes a two-stage AI pipeline combining Object Detection and Classification to automatically identify media spacer part numbers used in HDD manufacturing.

**Technologies**

- Python
- YOLO
- OpenCV
- TensorFlow

---

### FlowSyn

A manufacturing web application developed to support production workflow, inventory monitoring, and dashboard visualization for manufacturing operations.

**Technologies**

- ASP.NET Web Forms
- C#
- SQL Server
- Bootstrap
- JavaScript
