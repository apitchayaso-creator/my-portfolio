Overview

V-SPACE is an industrial AI inspection system developed to automatically identify spacer part numbers used in HDD manufacturing. The system combines object detection and image classification to improve inspection accuracy and reduce manual verification.

Problem

Operators needed to manually inspect spacer part numbers, which was time-consuming and susceptible to human error.

Solution

Developed a two-stage AI pipeline using object detection followed by classification to recognize spacer part numbers automatically.
